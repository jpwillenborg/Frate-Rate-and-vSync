\## Project Description

This is a small experiment using Unity's URP for 2D.  It applies the 2D Pixel Perfect package for the camera to keep everything aligned and crisp, and the Tilemap system for the graphics. Toggling the pixel cam UI on/off shows the effect, which is most noticable when moving the player and scrolling the background side to side. Shadows for the 2D lights were added as well as a simple respawning feature for when the player falls off a ledge. Also, a parallaxing effect was applied to the background along with a CRT-like canvas overlay "filter" which can be adjusted through the UI.

<br>



Note: Some assets from the Unity Asset Store are required for a complete build. They are not included in this repo. Please see the Licenses section below for links to the assets. Otherwise, feel free to browse through the project files. Thanks.

\## Player Controls

Move Player: A/D Keys -or- Left/Right Arrows

<br>

Jump: Spacebar

<br>

Double Jump: Spacebar x2

<br><br>



\## Project Features

\* 2D Pixel Perfect camera system

\* Tilemap and Palette system for graphics

\* 2D Light Shadow Casters

\* CRT-like canvas overlay

\* Respawning feature

\* Parallaxing background

\* Framerate counter
